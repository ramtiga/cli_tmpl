# CLI Templete

* Create CLI App Templete for ruby.


## Install

    $ git clone git://github.com/ramtiga/cli_tmpl.git
    $ cd cli_tmpl


## Usage

    $ ./bin/cli_tmpl -h
    Usage:
      cli_tmpl [app_name] [path]

    $ ./bin/cli_tmpl sample ./
    $ cd sample
    $ tree
    .
