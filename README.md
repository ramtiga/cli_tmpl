# CLI Templete

* Create CLI App Templete for ruby.


## Install

    $ git clone git://github.com/ramtiga/cli_tmpl.git
    $ cd cli_tmpl


## Usage

    $ ruby cli_tmpl.rb sample
    $ cd sample
    $ tree
    .
    |-- Gemfile
    |-- README.md
    |-- bin
    |   `-- sample
    `-- lib
        |-- sample
        |   |-- command.rb
        |   `-- version.rb
        `-- sample.rb
    
    3 directories, 6 files
